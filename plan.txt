## Stepped refactor plan (verify after each step)

Goal: every step is a **small, shippable change** with a **short manual smoke test** before the next step. Prefer **move code, don’t rewrite** until the monolith is split.

---

### Step 0 — Baseline (no code change)

**Do:** Note current behavior once (screenshots or a checklist).

**Smoke test checklist** (reuse after every step):
1. Map loads, hillshade visible  
2. **Auto mode:** pan → compute runs, sectors/contours/path-only each work  
3. **Manual mode:** add airport → **Run** → overlay + hover path  
4. **Sectors:** fill + grey borders + opacity slider  
5. **Cache mode:** select cell → Cache → Done  
6. Airspace toggle (if configured)  
7. No errors in browser console  

---

### Step 1 — Extract CSS (`index.html` → `styles.css`)

**Scope:** Move `<style>` block only; link `styles.css` in HTML.

**Risk:** Very low.

**Verify:** Page looks identical; params panel, tooltips, mobile layout unchanged.

---

### Step 2 — `js/dom.js` (DOM refs only)

**Scope:** Move all `document.getElementById(...)` (and similar) into `dom.js`; export one object e.g. `export const dom = { ... }`. `app.js` imports it.

**Risk:** Low — no logic move.

**Verify:** Same smoke test; especially buttons, sliders, mode toggle, help popover.

---

### Step 3 — `js/constants.js`

**Scope:** Move pure constants (`DEFAULT_MAX_ALTITUDE`, `MAP_CENTER`, layer IDs, `PARAM_HELP`, `VIZ_HINTS`, debounce ms, etc.).

**Risk:** Low.

**Verify:** Smoke test + confirm help text and viz hints still show.

---

### Step 4 — `js/app-state.js` (skeleton only)

**Scope:** Create `createApp()` returning an object with **all** current top-level `let` fields (initial values copied). `app.js` does `const app = createApp()` but **still uses local variables** OR aliases `let map = app.map` at top — minimal churn.

**Risk:** Low if you only introduce the object without rewiring every function yet.

**Verify:** Smoke test unchanged.

---

## Rules between steps

1. **One step = one PR/commit** (or one session).  
2. **No behavior change** in extract steps — copy-paste, export, import.  
3. **Don’t** split `app` state and extract modules in the same step until Step 4 exists.  
4. If something breaks, **revert that step only** — steps are ordered to keep diffs small.  
5. Optional later: add **one** Playwright smoke script after Step 7 (map load + mode select) — not required for manual checks.
